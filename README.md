# COVID-19 Global Data Analysis

This project analyzes the spread and impact of COVID-19 across 248 countries using the OWID (Our World in Data) dataset. The objective is to examine case counts, fatalities, CFR (Case Fatality Ratio), and timeline trends to understand pandemic behavior globally.
